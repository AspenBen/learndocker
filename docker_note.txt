Docker的三大组件：
镜像、容器、仓库

ctrl+P ctrl+Q可以退出容器但是不像exit会直接关闭容器，而是会使容器在后台运行

Docker run -d +容器id 可以使容器运行  并且进入容器内部

Docker logs +容器id：查看容器的运行情况

Docker top 显示最近的运行情况

Docker exec 在已经运行的容器中添加应用运行

Docker stop/kill stop会停止容器的运行 但是需要等待一段时间 kill可以直接终止容器的运行

注意使用Docker自带的手册学习Docker命令的详细介绍

容器所映射的端口和IP地址是一个虚拟的，并且是随机动态分配的，容器没启动一次 就会给分配一次

docker可以将一个容器导出为镜像
用户既可以使用 docker load 来导入镜像存储文件到本地镜像库，也可以使用 docker import 来导入一个容器快照到本地镜像库。这两者的区别在于容器快照文件将丢弃所有的历史记录和元数据信息（即仅保存容器当时的快照状态），而镜像存储文件将保存完整记录，体积也要大。此外，从容器快照文件导入时可以重新指定标签等元数据信息.
